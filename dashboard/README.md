# Aether-Chain Dashboard

Real-time blockchain explorer for Aether-Chain. Displays live block updates via WebSocket connection.

## Features

- **Live Block Stream** - New blocks appear instantly via WebSocket
- **Connection Status** - Visual indicator for network connectivity
- **Block Details** - Shows block ID, hash, and timestamp

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18
- Vite
- TailwindCSS
